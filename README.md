# 🎬 TMDB IPTV Generator

Ce projet génère automatiquement une **playlist IPTV VOD enrichie** (films + séries) à partir de l’API **TMDB**, puis met à jour le fichier `.m3u` chaque semaine via **GitHub Actions**.

---

## 🚀 Fonctionnalités
- Récupère les métadonnées TMDB (titre, année, synopsis, affiche).
- Génère un fichier `vod_enriched.m3u` compatible avec **TiviMate**.
- Mise à jour automatique chaque lundi à 10h UTC.
- 100 % gratuit et autonome.
